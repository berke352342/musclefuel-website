# MuscleFuel Website!


This project is a fitness supplement website which is called **MuscleFuel**, created for COMP 1112 Final Project (Phase 1)


## Features
- Home page, About, Shop, Product, and Contact pages
- Custom CSS design with cosisntent red/black/white theme
- Desktop first responsive layout
- Semantic HTML structures
- Two Google Fonts integrated


## Technologies
- HTML
- CSS
- Github for version control


## Setup Instructions
1. Clone the repository
2. Open `index.html` in your browser
3. Explore all pages via navigation bar
